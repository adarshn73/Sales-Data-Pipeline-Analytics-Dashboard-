📊 Sales Forecasting Dashboard (PulseFuel Nutrition)

Overview
This project analyzes sales performance for a simulated supplement company, PulseFuel Nutrition. The goal was to build a complete analytics pipeline: generate realistic sales data, clean and analyze it using Python, forecast future trends, and visualize insights through an interactive Power BI dashboard.

---

 Project Features
- Synthetic retail sales dataset (8,000+ transactions)
- Data cleaning and feature engineering using Python
- KPI analysis (Revenue, Profit, Units Sold, Orders)
- Product, category, and channel performance analysis
- Monthly sales trend analysis
- 3-month revenue forecasting model
- Interactive Power BI dashboard with filters and insights

---

Tools & Technologies
- Python (pandas, numpy, matplotlib)
- Power BI
- CSV data processing
- Basic forecasting (moving average model)

- Profit Margin

---

Key Insights
- A small number of products drive the majority of total revenue
- Sales show clear seasonal patterns, with spikes in January and summer months
- Direct-to-consumer (Website) channel likely provides stronger margins
- Certain regions consistently outperform others in revenue
- Sample packs act as entry-level products for customer acquisition

---
 Forecasting
A simple 3-month moving average model was used to forecast future revenue trends. The dashboard visualizes both historical and predicted data, clearly separating past performance from future projections.

---
 Business Impact
This dashboard can help businesses:
- Plan inventory for high-demand periods
- Identify top-performing products
- Optimize marketing strategies by region and channel
- Make data-driven decisions using forecast insights
